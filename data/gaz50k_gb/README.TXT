	   Notes on the Supply of Ordnance Survey Digital Data on CD-ROM
	   -------------------------------------------------------------

   Directory Structure of the CD-ROM
   ---------------------------------

   The directory structure of this CD-ROM is shown below:

				  ROOT 
				       |
				 ------------
				|            |
			   Data          Doc

   The ROOT directory will contain the following ASCII text file:
	  o This file - README.TXT.

   The Data directory will contain the data files for your order.

   The Doc directory will contain the following ASCII text files:
	  
	  o disccare.txt - information on the care of CDs

	  o data_licence.txt - information on the care of CDs
	  
   The Doc directory may also contain files specifically relating
	  to the current version of the product.
